Variables and Data Types

Numeric Data Types:

In Python, variables are used to store data values. The data type of a variable determines the type of data it can hold. Python supports several built-in data types, including:

  • int: Used to represent integer values.
  • float: Used to represent floating-point numbers with decimal places.
  • complex: Used to represent complex numbers in the form a + bj, where a and b are real numbers, and j is the imaginary unit.

    • Example:

              
  # Numeric data types
  age = 25  # int
  price = 9.99  # float
  complex_num = 2 + 3j  # complex

    String Data Type:

  • str: Used to represent sequences of characters enclosed in single quotes (' ') or double quotes (" ").

    • Example:

              
  # String data type
  name = "John Doe"  # str
  message = 'Hello, World!'  # str

    Boolean Data Type:

  • bool: Used to represent either True or False.

    • Example:

              
  # Boolean data type
  is_valid = True  # bool
  is_empty = False  # bool

    List Data Type:

  • list: Used to represent an ordered collection of elements enclosed in square brackets [ ].

    • Example:

              
  # List data type
  numbers = [1, 2, 3, 4, 5]  # list
  names = ["Alice", "Bob", "Charlie"]  # list

    Tuple Data Type:

  • tuple: Used to represent an ordered, immutable collection of elements enclosed in parentheses ( ).

    • Example:

              
  # Tuple data type
  coordinates = (10, 20)  # tuple
  colors = ("red", "green", "blue")  # tuple

    Dictionary Data Type:

  • dict: Used to represent a collection of key-value pairs enclosed in curly braces { }. Example:

Example:

          
  # Dictionary data type
  person = {"name": "John", "age": 30, "city": "New York"}  # dict
Control Flow

Control flow refers to the order in which statements are executed in a program. In Python, control flow is managed using conditional statements (if, elif, else) and looping statements (for, while).

  • Conditional Statements:
    •           
  # Example 1: if statement
  x = 10
  if x > 0:
      print("x is positive")

  # Example 2: if-else statement
  x = -5
  if x > 0:
      print("x is positive")
  else:
      print("x is non-positive")

  # Example 3: if-elif-else statement
  x = 0
  if x > 0:
      print("x is positive")
  elif x < 0:
      print("x is negative")
  else:
      print("x is zero")
  • Looping Statements:
    •             
  # Example 1: for loop
  fruits = ["apple", "banana", "cherry"]
  for fruit in fruits:
      print(fruit)

  # Example 2: while loop
  count = 0
  while count < 5:
      print(count)
      count += 1
  • Control Statements (break and continue):
    •             
  # Example 1: break statement
  fruits = ["apple", "banana", "cherry", "date"]
  for fruit in fruits:
      if fruit == "cherry":
          break
      print(fruit)

  # Example 2: continue statement
  numbers = [1, 2, 3, 4, 5]
  for number in numbers:
      if number % 2 == 0:
          continue
      print(number)
Functions

In Python, functions are reusable blocks of code that perform a specific task. They help in organizing code, improving code reusability, and promoting modular programming.

          
  # Example 1: Function without parameters
  def greet():
      print("Hello, there!")
      
  # Calling the function
  greet()  # Output: Hello, there!

In the above example, we define a function called greet() that doesn't take any parameters. When the function is called, it prints the message "Hello, there!".

          
  # Example 2: Function with parameters
  def greet(name):
      print("Hello, " + name + "!")
      
  # Calling the function with arguments
  greet("Alice")  # Output: Hello, Alice!
  greet("Bob")    # Output: Hello, Bob!

In this example, the function greet() takes a parameter name. When the function is called, we pass an argument to the function representing the name. The function then prints the personalized greeting using the provided name.

          
  # Example 3: Function with return statement
  def add_numbers(x, y):
      sum = x + y
      return sum
      
  # Calling the function and storing the result in a variable
  result = add_numbers(5, 3)
  print(result)  # Output: 8

In this example, the function add_numbers() takes two parameters x and y and returns their sum using the return statement. We call the function with arguments 5 and 3 and store the returned value in the variable result. Finally, we print the value of result, which is 8.

Modules

In Python, modules are files that contain Python code, including variables, functions, and classes. They allow you to organize and reuse code by breaking it into separate files.

      
  # Example: Importing a module
  import math
  Using functions from the math module
  print(math.sqrt(25)) # Output: 5.0
  print(math.pi) # Output: 3.141592653589793

In the above example, we import the math module using the import statement. The math module provides various mathematical functions and constants. We can use the functions from the math module by prefixing them with math..


  Example: Importing specific functions from a module
  from random import randint, choice

  Using the imported functions
  print(randint(1, 10)) # Output: Random integer between 1 and 10
  print(choice(['apple', 'banana'])) # Output: Randomly chosen element from the list

In this example, we use the from module import function syntax to import specific functions from the random module. This allows us to use the functions directly without the need to prefix the module name.

You can also create your own modules by defining Python code in separate files and importing them into your programs.


  Example: Creating and using a custom module
  File: mymodule.py
  def greet(name):
  print("Hello, " + name + "!")

  File: main.py
  import mymodule

  mymodule.greet("Alice") # Output: Hello, Alice!

In this example, we have a custom module called mymodule.py that contains a function greet(). We import the mymodule module into the main.py file and use the greet() function to print a greeting.

File Handling

In Python, file handling allows you to work with files, such as reading from or writing to them. It enables you to perform tasks like opening files, reading their contents, writing data to them, and closing them when you're done.

Here are some examples of common file handling operations:

      
  # Example: Opening and reading a file
  file = open("data.txt", "r")  # Open file in read mode
  Read the contents of the file
  contents = file.read()

  Print the contents
  print(contents)

  Close the file
  file.close()

In the above example, we open a file named "data.txt" in read mode using the open() function. We read the contents of the file using the read() method and store them in the contents variable. Finally, we print the contents and close the file using the close() method.


  Example: Writing to a file
  file = open("output.txt", "w") # Open file in write mode

  Write data to the file
  file.write("Hello, world!")

  Close the file
  file.close()

In this example, we open a file named "output.txt" in write mode. We write the string "Hello, world!" to the file using the write() method. Finally, we close the file.

It's important to note that when opening a file, you should always close it after you're done with it to free up system resources. Alternatively, you can use the with statement, which automatically takes care of closing the file for you:


  Example: Using the with statement for file handling
  with open("data.txt", "r") as file:
  contents = file.read()
  print(contents)

In this example, the with statement is used to open the file "data.txt" in read mode. The file is automatically closed at the end of the block, even if an exception occurs.

File handling in Python offers various methods and modes for different file operations, such as reading line by line, appending to a file, and more. You can explore the Python documentation for more details on file handling operations.

Exception Handling

In Python, exception handling allows you to handle and recover from errors or exceptional situations that may occur during the execution of your code. By handling exceptions, you can prevent your program from crashing and provide alternative paths or error messages.

Here's an example of how exception handling works:

      
  # Example: Handling a specific exception
  try:
      # Code that may raise an exception
      result = 10 / 0  # Raises a ZeroDivisionError
  except ZeroDivisionError:
      # Code to handle the specific exception
      print("Cannot divide by zero!")

In the above example, we have a try block that contains the code that may raise an exception. In this case, we attempt to divide the number 10 by zero, which raises a ZeroDivisionError. We use the except block to specify the exception we want to handle. In this case, we handle the ZeroDivisionError and print a custom error message. 

      
  # Example: Handling multiple exceptions
  try:
      # Code that may raise exceptions
      result = int("abc")  # Raises a ValueError
      result = 10 / 0     # Raises a ZeroDivisionError
  except ValueError:
      # Code to handle ValueError
      print("Invalid input!")
  except ZeroDivisionError:
      # Code to handle ZeroDivisionError
      print("Cannot divide by zero!")

In this example, we have multiple except blocks to handle different exceptions. The code inside the try block attempts to convert the string "abc" to an integer, which raises a ValueError. We also attempt to divide the number 10 by zero, which raises a ZeroDivisionError. Each exception is handled separately, and the appropriate error message is printed.

In addition to handling specific exceptions, you can use a generic except block to catch any exception that is not explicitly handled:

      
  # Example: Handling any exception
  try:
      # Code that may raise an exception
      result = 10 / 0  # Raises a ZeroDivisionError
  except Exception as e:
      # Code to handle any exception
      print("An error occurred:", str(e))

In this example, the except block with the Exception keyword is used to handle any exception that may occur. The as e part allows you to access the exception object and obtain information about the error. In this case, we print a generic error message along with the error details.

Exception handling in Python provides flexibility in handling different types of errors and implementing appropriate error-handling logic. It helps in creating robust and reliable code that gracefully handles unexpected situations.

Classes and Objects

In Python, a class is a blueprint for creating objects. It defines the properties and behaviors that an object of that class will have. An object is an instance of a class, and it can have its own unique data and functionality.

Here's an example of a class and its object:

      
    # Example: Creating a class
    class Car:
        def __init__(self, make, model, year):
            self.make = make
            self.model = model
            self.year = year

        def drive(self):
            print(f"Driving the {self.make} {self.model}!")

    Creating an object of the Car class
    my_car = Car("Tesla", "Model S", 2022)

    Accessing object properties
    print(my_car.make) # Output: Tesla
    print(my_car.model) # Output: Model S
    print(my_car.year) # Output: 2022

    Calling object methods
    my_car.drive() # Output: Driving the Tesla Model S!

In the above example, we define a class called Car using the class keyword. The Car class has three properties: make, model, and year. The init() method is a special method called the constructor, which is executed when a new object is created. It initializes the object's properties.

The drive() method is a function defined within the class. It can be called on an object of the Car class and performs a specific action.

We create an object my_car of the Car class by invoking the class as if it were a function, passing the required arguments to the constructor.

We can access the properties of the object using the dot notation, such as my_car.make, my_car.model, and my_car.year. We can also call the methods of the object, such as my_car.drive().

Classes and objects provide a way to organize and encapsulate data and related functionality, enabling the creation of complex systems and code reusability.

Libraries and Packages

In Python, libraries and packages are essential for extending the functionality of the language. They provide pre-written code and modules that can be imported into your programs, saving you time and effort in implementing common tasks and functionalities.

A library is a collection of modules, while a module is a single file containing Python code. Modules can contain functions, classes, variables, and other objects that can be used in your program.

Python's standard library comes with a wide range of modules that cover various areas, such as math, file I/O, networking, and more. These modules are readily available for use and can be imported using the import statement.

Here's an example of importing and using a module from the standard library:

      
  # Example: Using the math module
  import math
  Using functions from the math module
  print(math.sqrt(25)) # Output: 5.0
  print(math.sin(math.pi/2)) # Output: 1.0

In this example, we import the math module using the import statement. We can then access the functions and constants provided by the module using the dot notation, such as math.sqrt() and math.sin().

In addition to the standard library, there are numerous third-party libraries and packages available for Python. These libraries are developed by the Python community and cover a wide range of domains, such as web development, data analysis, machine learning, and more.

To use third-party libraries, you need to install them using a package manager like pip and then import them into your program. Here's an example:


  Example: Using a third-party library
  import requests

  Making a HTTP GET request using the requests library
  response = requests.get('https://api.example.com/data')
  print(response.status_code) # Output: 200

In this example, we import the requests library, which is a popular library for making HTTP requests. We use the requests.get() function to send a GET request to a specified URL and store the response in the response variable.

Libraries and packages greatly enhance the capabilities of Python, allowing you to leverage existing code and solutions to build powerful and efficient applications.
